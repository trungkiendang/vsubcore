Powerful speech recognition tool for converting video/audio to text and generating SRT subtitles

About

VSubCore is a powerful speech recognition tool developed by Trung Kien Dang. It helps you convert video/audio files to text and generate SRT subtitles with support for multiple languages.

Features

Convert Media

Convert video/audio to text

Support multiple languages

Generate SRT subtitles

Fast and efficient processing

Installation

pip install vsubcore

Usage

vsub-nolog.py source_path [-o output] [-S stt-language]

Example:

vsub-nolog.py video.mp4 -o subtitles.srt -S en

Getting Started

  1. Clone the repository: 
    git clone https://github.com/trungkiendang/vsubcore.git
  2. Install dependencies: 
    pip install -r requirements.txt
  3. Run the tool: 
    python vsub-nolog.py your_video.mp4
